# .github

[![License](https://img.shields.io/github/license/VecTrade-io/.github)](LICENSE)

Organization-level community health files and CI workflows for [VecTrade](https://github.com/VecTrade-io).

## Contents

| File | Purpose |
|------|---------|
| `profile/README.md` | Organization profile (shown on github.com/VecTrade-io) |
| `CONTRIBUTING.md` | Default contributing guidelines for all repos |
| `CODE_OF_CONDUCT.md` | Community code of conduct |
| `SECURITY.md` | Vulnerability reporting policy |
| `ISSUE_TEMPLATE/` | Default issue templates |
| `PULL_REQUEST_TEMPLATE.md` | Default PR template |
| `FUNDING.yml` | Sponsorship configuration |
| `workflows/` | Reusable CI/CD workflows |

These files are automatically inherited by all repositories in the VecTrade-io organization that don't define their own.
