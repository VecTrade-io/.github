VecTrade provides a unified API for financial market data, analysis, and AI-powered insights. Build trading tools, dashboards, and fintech applications with our SDKs, CLI, and MCP integration.

### SDKs & Tools

| Package | Description |
|---------|-------------|
| [`vectrade-python`](https://github.com/VecTrade-io/vectrade-python) | Python SDK |
| [`vectrade-node`](https://github.com/VecTrade-io/vectrade-node) | TypeScript/Node.js SDK |
| [`vectrade-cli`](https://github.com/VecTrade-io/vectrade-cli) | Command-line interface |
| [`vectrade-mcp`](https://github.com/VecTrade-io/vectrade-mcp) | Model Context Protocol server |
| [`vectrade-ai-provider`](https://github.com/VecTrade-io/vectrade-ai-provider) | Vercel AI SDK provider |
| [`finkit`](https://github.com/VecTrade-io/finkit) | Technical analysis library |
